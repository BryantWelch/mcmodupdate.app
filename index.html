Welcome to the Minecraft Mod Updater & Builder!

This tool is your one-stop shop for managing your Minecraft mods. You can easily find updates for your entire mod list, discover new mods, and export a ready-to-use modpack for your favorite launcher.

Ready to Start?

To get started, you can upload an entire modpack (.mrpack), individual mods or plugins (.jar), resource packs, shaders, or data packs (.zip), import a previously exported modlist (.json), or use the search bar to find new content. Then, click the "Update" button to see the magic happen!

Minecraft Mod Updater & Builder

This tool helps you update your existing Minecraft mods or build a brand new mod list from scratch. It uses the Modrinth API to find the latest compatible versions and can export your list as a standard Modrinth modpack. Note: This tool only searches Modrinth - mods from CurseForge or other sources won't be found. Dependencies are automatically detected and added to ensure your modpack works properly.

How to Use:

  • Loader / Version: Select your desired mod loader (Fabric, Forge, NeoForge, Quilt) and target Minecraft version.
  • Upload Files: Drop a .mrpack file to load a modpack, or drag and drop your local .jar (mods, plugins) and .zip (resource packs, shaders, data packs) files into the upload area. You can also drop a previously exported .json modlist to restore it.
  • Search for New Mods: Use the search bar to find and add mods directly from Modrinth.
  • Process Your List: Click "Update" to find the latest compatible versions and dependencies.

Understanding Results:

Color-Coded Borders:

  • Green border: Successfully found updates for these mods
  • Blue border: Dependencies automatically added to ensure compatibility
  • White border: Mods already up-to-date with the latest version
  • Yellow border: No compatible version found for your selected loader/version
  • Red border: Mod not found on Modrinth or file could not be processed

Icons Next to Mod Titles:

  • Box: Standard mod
  • Paintbrush: Resource pack
  • Glasses: Shader pack
  • Brackets: Data pack
  • Open Box: Modpack
  • Plug: Plugin
  • Puzzle Piece: This mod is required by another mod in your list
  • Chain Link: This mod has dependencies that were automatically added
  • Lightning Bolt: Data loaded from cache for faster performance
  • Double Checkmark: This mod is already up-to-date with the version selected

Features & Tools:

  • Search & Filter: Use the search boxes to quickly find specific mods in your results
  • Sort Options: Click the sort button to organize by name, downloads, followers, or filter by environment/channel
  • Copy Lists: Use the copy button to quickly copy mod names to your clipboard
  • Export Lists: Use the export button to download the current list as CSV, JSON, Markdown, or plain text. The JSON format is the one you can re-import — CSV/Markdown/plain text are share-oriented and not importable.
  • Import Modlist: Drag and drop a previously exported .json file onto the upload area to restore that modlist.
  • Version Selection: Choose between Release, Beta, or Alpha versions for each mod
  • Download Options: Download individual mods or get everything as a ready-to-use .mrpack file

Attributions:

This tool relies on the Modrinth API for all mod data. Please be aware of their ratelimits. The ratelimit is currently 300 requests per minute. The application will automatically pause and resume to respect these limits.

Privacy Policy

Minecraft Mod Updater is a static, client-side web application. There is no backend server collecting or storing data about you. Everything happens locally inside your browser tab.

Your Files

Any .jar, .zip, .mrpack, or .json files you upload are processed entirely in your browser. They are never uploaded to mcmodupdate.app or any third party. When you close or refresh the tab, the files are gone.

Browser Storage

To improve performance, the application stores a small amount of non-personal data locally on your device:

  • localStorage — caches the list of Minecraft versions and Modrinth platform statistics so the app loads faster on subsequent visits.
  • sessionStorage — caches mod lookups during your current tab session and is cleared automatically when you close the tab.

You can clear this data at any time using your browser's site-data controls. The site does not use tracking cookies.

Third-Party Services

The site loads resources from the following third parties. Each may receive standard request data (your IP address, user-agent, and the resource being requested) under their own privacy policies:

  • Modrinth API (api.modrinth.com) — used to search for mods and retrieve version and dependency information.
  • Modrinth CDN — serves mod icons and gallery images.
  • jsDelivr, unpkg, cdnjs — public JavaScript content delivery networks used to load the libraries the app depends on (JSZip, toml-js, marked, crypto-js).
  • Ko-fi — only contacted when you click the "Buy Me a Coffee" button.

What We Do Not Do

  • No user accounts, sign-ups, or logins.
  • No first-party analytics or fingerprinting scripts.
  • No tracking cookies.
  • No advertising or ad-network integrations.
  • No selling, sharing, or transferring of any user data — because we don't collect any.

Open Source

The complete source code is available on GitHub, so you can verify the privacy claims above. If you find something that contradicts this policy, please open an issue.
